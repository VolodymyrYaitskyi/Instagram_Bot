This bot created for personal use

Can to like posts of my subscribers and to like posts by hashtag in google chrome
saving statistics of number of likied posts using selenium, pynput, sqlite3 libraries

To run this app for your account you should:
1)Have installed selenium, pynput, sqlite3 libraries
2)set path to your Google\Chrome\User Data at  "options.add_argument()"
3)set path to your chromedriver.exe 
4)set path to your subscribers at "my_acc_url"

Run bot with one of the methods: my_bot.like_photo_by_hashtag('nikonphoto', 50) #1st argument is hashtag to like, 2nd argument is number of posts to like
my_bot.like_my_subscribers(3) # argument is number of posts to like
